<p>
    <strong>Task:</strong><br>
    Необходимо автоматизировать рутинный процесс заполнения данных BSS (Lac, Rac, Bsc, Bcf) Nokia, Ericsson и Булат на сайте createsite.<br>
    <strong>Decision:</strong><br>
    для работы с проектом использую скрипты bat.<br>
    Необходимо скачать Python (У меня 3.10) и установить ее. Затем уже использовать скрипты:<br>
    - checkPython.bat для проверки Python<br>
    - checkParsing.bat для настройки переменного окружения и создания проекта Parser<br>
    - checkYandex.bat для проверки Яндекса<br>
    - addYandexDriver.bat для настройки Яндекс драйвера<br>
    По умолчанию проект устаановится в домашнюю директорию. при желании можно переименовать свою папку в файле checkParsing.bat &nbsp;(строка 9: SET VENV_NAME=env*). файл покажет какой командой звпускать переменную окруженния и куда установил исходные файлы скрипта. файл requirements.txt должен находиться в той же папке, в которой вы звпускаете скрипт.<br>
    Алгоритм:<br>
    - Вся информация, которая отображается на сайте хранится в БД mysql. Есть права на чтение БД. Поэтому необохидимо найти в БД BSS часть, в которой отсутсвуют данные и загрузить первый отчет библиотекой pandas в виде таблице.<br>
    - Загрузить второй отчет в виде таблицы с данными, которые уже заполнена BSS часть. Также в этот отчет включить информацию по координатам БС.<br>
    - Сравнить два отчета с целью определить является БС новым или довесом.<br>
    - если является довесом, то автоматически заберет данные BSS из второго отчета и записать в третий отчет в виде таблицы.<br>
    - если это новый БС, тогда нужно распарсить сайт rdb библиотекой Selenium, в котором хранится информация (Координаты и УЦН) о новом БС и записать полченные данные в четвертый отчет в виде таблицы.<br>
    - Если у БС есть информация про наличия УЦН, то автомтически подставить данные BSS.<br>
    - Сравнить координатами четвертый отчет со вторым определния ближайщих соседств и записать полученные данные BSS в пятый отчет в виде таблицы.<br>
    - Содеинить третий отчет, в которым были записаны довесы, с пятым отчетом с данными о новых БС и записать в excel файл.<br>
    - Доступа к БД Mmysql с разрешением на добавление данных в БД отствуют, поэтому библиоткой selenium импортировать полученные файлы в createsite.<br>
    - написать отдельный скрипт shell, который запускает весь процесс проекта python.<br>
    - Настроить планировщик задач, чтобы по расписанию запускался shell скрипт.<br>
    От проекта зависят файлы:<br>
    - config.txt - используется для настройки доступа (БД, доступ к сайтам) при работе с парсингом сайта<br>
    - addBssCes.py - основной код проекта<br>
    - startBssCes.ps1 - для запуска проекта<br>
    Перед запуском необходимо заполнить файл config.txt.<br>
    в файле startBssCes.ps1 нужно заменить переменные в строках 1 и 2 на актуальные данные. файл будет записывать в лог файл всю процедуру запуска. скрипт можно запустить в планировщике задач.<br>
    в addBssCes.py нужно менять параметры подключения яндекс драйвера<br>
    в скрипте checkYandex.bat нужно для параметров в реестре поменять переменные в строках:<br>
    SET RegKey="*"<br>
    SET RegValue1=*<br>
    SET RegValue2=*<br>
    Скрипт покажет место установки и версию яндекс.<br>
    Инструкции по поиску версии в реестре:<br>
    - Откройте редактор реестра:<br>
    - В левой панели редактора реестра перейдите по следующему пути:<br>
    HKEY_CURRENT_USER\Software\Yandex\YandexBrowser\BLBeacon<br>
    - В правой части окна найдите параметр с именем version. Значение этого параметра будет содержать установленную версию Яндекс.Браузера.<br>
    YandexDriver необохимо обновлять, поэтому в рабочей папке должен быть файл yandexdriver.exe. в скрипте addYandexDriver.bat нужно поменять параметр в строке 3:<br>
    SET PROJECT_NAME=*
